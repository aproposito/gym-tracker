# Gym Tracker

PWA estatica para anotar entrenos de gimnasio desde el movil.

Incluye selector de Rutina A, Rutina B y Rutina C, cada una con ejercicios propios editables.

## Uso local

Abre la carpeta con un servidor local y visita `http://localhost:8000`.

```bash
python3 -m http.server 8000
```

## Publicar en GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube estos archivos a la rama principal.
3. Activa GitHub Pages desde Settings > Pages.
4. Abre la URL publicada en Safari del iPhone.
5. Usa Compartir > Anadir a pantalla de inicio.

Los datos se guardan en el navegador del dispositivo. Usa exportar/importar JSON como copia de seguridad.
